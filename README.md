# Persona Chatbot

Simple persona-driven chat demo. System prompt switches between personas; model and API usage remain unchanged.

Backend: `backend/` (Express + Gemini)
Frontend: `client/` (Next.js app dir)

Quick start (backend):

```
cd backend
npm install
cp .env.example .env
npm run dev
```

Quick start (frontend):

```
cd client
npm install
npm run dev
```
